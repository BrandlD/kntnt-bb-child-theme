THE USE OF THE CUSTOM DIRECTORY
===============================

As soon as this theme is enabled, the following files are created for the
specified purpose.

custom.css
----------
This file replaces style.css as the place where you can add your own CSS to be
included with the theme.

custom.php
----------
This file replaces function.php as the place where you can add your own PHP
code to be executed when this theme is used.

custom.js
---------
This file plays the same role for JavaScript as function.php plays for PHP.
That is a place where you can add your own JavaScrit to obe loaded with the
theme.
